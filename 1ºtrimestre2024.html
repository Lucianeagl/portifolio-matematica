Sistema Decimal:

O sistema decimal tabém conhecido como sistema de numeração indo-arábico é o sistema ultilizado pelo mundo inteiro nos dias atuais, se utiliza a base 10, ou seja, se utiliza 10 símbolos diferentes para representar todos os números

Sistema Octal:

O sistema Octal é um sistema de numeração de base 8, ou seja utliza 8 digitos (0,1,2,3,4,5,6,7)


Sistema hexadecimal:

No sistema hexadecimalse utiliza todos numeros do sistema decimal e as letras A,B,C,D,E,F
